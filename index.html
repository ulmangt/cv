Open Source

Glimpse Data Visualization Library

Glimpse is a Java/OpenGL data visualization library designed to simplify utilizing powerful GPU hardware features like shaders, vertex buffers, and cached texture memory to display and interact with large data sets. It's available on Maven Central or on GitHub at metsci/glimpse.

Java Very Fast Machine Learning Toolkit

Based on original work by Hulten, G. and Domingos, P. "VFML -- A toolkit for mining high-speed time-changing data streams" http://www.cs.washington.edu/dm/vfml/. 2003. This software provides a Java implementation of the stream-based decision tree algorithm described by Hulten and Domingos. The algorithm utilizes the Chernoff (or Hoeffding) Bound to provide a statistical basis for when to branch the decision tree without having to re-analyze all available training examples. See: Mining Massive Data Streams.pptx or the JVMFL site for further details.

Malaysian Airlines Flight 370

Metron, Inc. was involved with the search for Air France Flight 447 and representatives appeared on a number of news segments following the Malaysian Airlines Flight 370 disappearance. One of those segments featured software which I developed; displaying global shipping patterns (or lack thereof) in the suspected region of the disappearance: Dangerous Conditions Await Search Crews.

Tactical Advancements for the Next Generation (TANG) Forum

In 2012 through 2014, the Navy hosted a series of design workshops aimed at developing the next generation of displays and controls for submarine combat systems. I constructed a number of demonstration visualization and data processing applications for the TANG tech expo. These demonstrations included iPad-based mapping applications (see minute 2:31 in the TANG video), a Google Glass Mirror API app, and interactive/hierarchical OpenGL-based Digital Navigation Charts.

Academic

JCUDA Dynamic Histogram Visualization

This project was an experiment in interfacing CUDA directly with Glimpse OpenGL visualizations. In this case, using CUDA to read heat map data stored in an OpenGL texture and calculate an on-the-fly histogram of a selected subset of the data.

Support Vector Machine Handwriting Recognition

AMPL was used to model a support vector machine optimization problem to classify low resolution images of the digits 0 through 9. Results, codes, and screenshots are available in the final report or the project site.
